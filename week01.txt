 git-scm.com 사이트 들어가서 x64 Setup (윈도우버전) 다운로드 후 설치하기
바탕화면에서 새폴더 만들고, 새폴더에 마우스 대고 오른쪽 버튼, 추가옵션표시 클릭 후 Open git bash here

* touch 파일명 (파일 생성하기)
폴더에서 ... 옆에 보기 / 표시 / 파일확장명 클릭
* ls (현재 있는 파일 확인하기)
txt메모장 열고 내용 입력 후
* cat 파일명 (파일 속 내용보기)

* ls -l (파일들이 생성된 시각 정보보기)

* git status (현재 파일들의 깃 여부 상태보기)
* git init (현재 이 파일을 깃으로 지정하기)
하면 .git 이라는 폴더가 또 생긴다 = 클릭하기

* ls
* ls -a (숨긴거보기)
폴더에서 ... 옆에 보기 / 표시 / 숨긴항목 클릭

* cd 파일명 (현재 폴더에서 지정한 하위폴더로 이동) 예를들어 cd .git
* ls
* cd .. (현재 폴더에서 바로 상위폴더로 이동)
* ls
* git status

. / .. / week01.txt 였던게 . / .. / .git / week01.txt 로 변함

* ls -a (파일로 가는 상위항목들 보기)
* ls -al (상위항목들까지 포함한 생성된 시각 정보보기)
