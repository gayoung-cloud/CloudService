* history : 그동안 이곳에 입력한 명령어 내역들 주루룩 보기

이어서

* git add week01.txt : 스테이지에 이 파일 올리기 = 이 파일 관리해
* git status 

* git add . : 스테이지에 현재 모든 파일 올리기
* git status : ALL녹색되고 이건 관리한다는 뜻

*** " . "은 현재폴더를 의미함

* git commit -m "커밋메세지" : 스테이지에 올린 파일들을 커밋 기록하기 = 해시코드가 부여되고 커밋한 아이디나 이메일을 남김(커밋에 함께 저장됨)
* git config~~~~복붙입력

* git log : 커밋 기록보기 ( 노란색은 해시코드임 )
해시코드 앞 7자 복사하기 : Enter키 

* git checkout 해시코드붙여넣기 : 세이브 포인트로 돌아가기
